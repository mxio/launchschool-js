**Convert the person object into a nested array nestedPerson, containing the same key-value pairs.**

```
let person = {
  title: 'Duke',
  name: 'Nukem',
  age: 33
};

let nestedPerson = Object.entries(person);
nestedPerson;

// Expected output:
// [['title', 'Duke'], ['name', 'Nukem'], ['age', 33]]
```
